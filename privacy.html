Overview

Content Machine is committed to protecting user privacy and ensuring secure handling of account data used within the platform.

This application is designed to provide social media automation, content management, workflow integration, and publishing services through authorized third-party platforms such as TikTok, Instagram, YouTube, Telegram, and related APIs.

Information We Access

When users connect their accounts through OAuth or authorized integrations, the application may access limited account information necessary for platform functionality, including:

  • Basic profile information
  • Access tokens and authentication data
  • Media publishing permissions
  • Analytics and engagement data
  • Account identifiers required for API communication

How Information Is Used

The collected information is used exclusively for:

  • Secure authentication
  • Automated content publishing
  • Workflow execution

  • Media scheduling
